``mx.apply``
========================

Description
----------------------

Apply symbol to the inputs.

Usage
----------

.. code:: r

	mx.apply(x, ...)

Arguments
------------------

+----------------------------------------+------------------------------------------------------------+
| Argument                               | Description                                                |
+========================================+============================================================+
| ``x``                                  | The symbol to be applied                                   |
+----------------------------------------+------------------------------------------------------------+
| ``kwargs``                             | The keyword arguments to the symbol                        |
+----------------------------------------+------------------------------------------------------------+
